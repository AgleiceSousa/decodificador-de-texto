# Decodificador de Criptografia de César

Este projeto é um decodificador de texto baseado na Criptografia de César.
Foi desenvolvido como parte de um desafio do curso de HTML, JavaScript e CSS
da Alura em parceria com a ONE GF.

## Sobre a Criptografia de César

A Criptografia de César é uma das técnicas de criptografia mais simples e
conhecidas. É um tipo de substituição na qual cada letra do texto é substituída
por outra, que se apresenta no alfabeto abaixo dela um número fixo de vezes.

## Como usar

1. Abra o arquivo `index.html` no seu navegador.
2. Insira o texto que você deseja criptografar ou descriptografar no campo de
entrada. Você pode digitar o texto diretamente ou copiar e colar o texto.
3. Selecione a ação que deseja realizar: "Criptografar" ou "Descriptografar".
4. Clique no botão correspondente para executar a ação.
5. O texto criptografado ou descriptografado será exibido.
